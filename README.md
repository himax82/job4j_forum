## О проекте
Проект представляет собой классический форум.

## Функционал
- Авторизация и регистрация
- Создание тем 
- Редактирование тем
- Просмотр созданных тем
- Создание сообщений в темах

## Используемые технологии
- Spring (Boot, Security, Data)
- JSP/JSTL/Bootstrap
- PostgreSQL
- Junit, Mockito
- Liquibase
- Java 14
- Maven
